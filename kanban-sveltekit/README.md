# Kanban SvelteKit

Kanban board built with SvelteKit, form actions, and SQLite.

## Setup

```bash
npm install
```

## Database

Initialize and seed the database:

```bash
npx drizzle-kit push
npm run seed
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```
