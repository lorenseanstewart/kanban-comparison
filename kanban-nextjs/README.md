# Kanban Next.js

Kanban board built with Next.js 15, Server Actions, and SQLite.

## Setup

```bash
npm install
```

## Database

Initialize and seed the database:

```bash
npx drizzle-kit push
npm run seed
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
